## Getting Started(JDBC 도서 관리 시스템)
MySQL과 Java JDBC를 활용한 도서 관리 시스템.

## 환경
언어: Java(JDK 21 이상 권장)
DB: MySQL 9.6.0
라이브러리: mysql-connector-j-9.6.0, lombok-1.18.42


## 주요 기능
1. 회원 관리: 회원 관련 CRUD 구현
2. 도서 관리: 도서 관련 CRUD 구현
3. 대출 관리: 도서 대출 및 반납, 개인 및 전체 대출 이력 조회
4. 예외 처리: 데이터 입력 타입 및 DB 관련 방어막 구축
        - 데이터 형식, 존재 하지 않는 데이터에 대한 예외 처리
        - DB가 연결되어 있지 않을 시, 프로그램을 강제 종료시키지 않고 초기 메뉴를 유지해줌.

## DB 스키마
해당 테스트 DB를 구축하기 위한 SQL문은 별도로 첨부함.

## 시작하는 법
1. DB 설정: src/dbutil/DBUtil.java 파일에서 본인의 MySQL url, username, password를 수정.
2. 드라이버 로드: lib 폴더에 ## 환경에서 언급된 라이브러리를 버전에 맞추어 설치.
3. 실행: App의 main()을 구동.

## 주의사항
.gitignore 설정에 의해 JDBC 드라이버(.jar)는 포함되어 있지 않음. 테스트 유저 개인적으로 라이브러리 추가 필요.
DB 서버가 실행 중이어야 모든 기능을 정상적으로 이용 가능.
